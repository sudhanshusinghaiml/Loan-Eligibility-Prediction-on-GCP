Project objective:

To build an automated machine learning model to predict the repaying capacity of applicants using Flask, Docker, Cloud Build, Cloud Run and Cloud Source Repository on GCP. The model predicts whether it is suitable to offer loan to the applicant or not
Description:
Data Description: The dataset contains information about 1,00,000 banking customers who have applied for the loan in the past months with various attributes such as Loan ID, CustomerId, Current Loan Amount, Term, Credit Score, Years in current job, Home Ownership, Annual Income, Purpose, Monthly Debt, Years of Credit History, Months since last delinquent, Number of Open Accounts, Number of Credit Problems, Current Credit Balance, Maximum Open Credit, Bankruptcies, Tax Liens etc.
Tech Stack:
Language: Python
Libraries: pandas, numpy, scipy, matplotlib, seaborn, sklearn, Flask, Docker, Ensemble, Gunicorn, Flask, Cloud Build, Cloud Run, Cloud Source Repository, joblib/pickle, GCP
Approach followed:
    1. Data Preprocessing:
  • Data Cleaning
  • Continuous Data Cleaning
  • Categorical Data Cleaning
  • Univariate Data Analysis
  • Multivariate Data Analysis
  • Missing Value Treatment
  • Outlier Treatment
  • Categorical Encoding
    2. Machine Learning Model Building:
  • Logistic Regression
  • Decision Tree Classification
  • Random Forest Classification
  • Gaussian Naive Bayes
  • Extreme Gradient Boosting Classification
  • Synthetic Minority OverSampling TEchnique
  • Applying Threshold to Improve Model Performance
    3. Model Deployment:
  • APIs - Application Programming Interface
  • Webdevelopment using Flask
  • GCP Deployment
  • Model Inference Pipeline
